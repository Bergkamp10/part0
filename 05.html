0.5 - Events when user visits SPA version

  1. Browser-->Server : HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa
  2. Server-->Browser : HTML-code
  3. Browser-->Server : HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.css
  4. Server-->Browser : main.css
  5. Browser-->Server : HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa.js
  6. Server-->Browser : spa.js

note over browser:
Browser receives spa.js and executes it. spa.js instructs the browser to make a
request for JSON-data from the server.

note over browser:
When the browser receives the JSON-data, it continues executing the javascript code,
building the HTML to present the notes using the DOM.
