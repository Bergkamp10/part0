0.6 - Events when user clicks submit in SPA

  1. Browser-->Server : HTTP POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa

note over server:
Server receives form data and perform server-side operations (saving to database).
Server then returns response with status code 201 (resource created)

note over browser:
browser does not make any more requests here, but uses the DOM to present the latest note
on the frontend (it does not fetch it from the database)
